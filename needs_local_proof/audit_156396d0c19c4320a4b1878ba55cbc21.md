## Attacker Path
**preconditions:**
- A token is in `TokenStatus.Tradable` with `circulatingSupply` just below `dexSupplyThresh`
- The migration trigger lives inside `PORTAL_TRADE_V2` (not in this repo's indexed source)
- The critical assumption: `PORTAL_TRADE_V2` calls the external migrator contract **before** calling `_setTokenStatus(DEX)` (CEI violation)

**attacker-controlled inputs:**
- `ExactInputParams` with `inputToken` = quote token, `outputToken` = target bonding-curve token, `amountIn` = exact amount to push `circulatingSupply` past `dexSupplyThresh`
- A malicious `recipient` or migrator callback contract that re-enters `swapExactInput`

**call sequence:**
1. Attacker calls `swapExactInput` with exact amount to trigger migration
2. Inside `PORTAL_TRADE_V2`, migration threshold is crossed; migrator is called externally
3. **If** `_setTokenStatus(DEX)` has not yet been called at this point, `state.status` is still `Tradable`
4. Migrator callback re-enters `swapExactInput`
5. `_getSwapImplementation` reads storage: `state.status == Tradable` → routes to `PORTAL_TRADE_V2`
6. Attacker buys tokens at bonding-curve price; `circulatingSupply` increases further
7. Outer call resumes; migrator receives less reserve than expected (or migration proceeds with corrupted state)

## Why Existing Checks Fail
**No reentrancy guard anywhere in `src/`**: A grep for `nonReentrant`, `ReentrancyGuard`, `_reentrancyGuard`, and `_status` across all `src/**/*.sol` returns zero matches. [1](#0-0) 

**Status-based routing is the only natural guard**: `_getSwapImplementation` re-reads `state.status` from storage on every call. If `_setTokenStatus(DEX)` is written **before** the external migrator call (CEI-compliant), a reentrant `swapExactInput` would see `TokenStatus.DEX` and be routed to `PORTAL_DEX_ROUTER` — the unfunded DEX pool would revert, blocking the attack. But if `PORTAL_TRADE_V2` calls the migrator first and sets status after (CEI violation), the window is open. [4](#0-3) 

**Critical gap**: `PORTAL_TRADE_V2` (the facet that contains the actual buy/migration logic, `dexSupplyThresh` comparison, and the ordering of `_setTokenStatus` vs external migrator call) is **not present in this repository**. The repository contains only `Portal.sol`, `PortalBase.sol`, `PortalCommon.sol`, and interfaces. The entire attack hinges on whether `PORTAL_TRADE_V2` violates CEI — this cannot be confirmed from the indexed source alone. [5](#0-4) 

## Rejection Checks
**expected behavior checked:** The status-based routing in `_getSwapImplementation` is a natural (if implicit) reentrancy guard — but only if CEI is followed in `PORTAL_TRADE_V2`. Not documented as an intentional guard.

**prior report checked:** Not determinable from available source.

**README/NatSpec checked:** No NatSpec on `swapExactInput` or `_getSwapImplementation` addresses reentrancy or migration ordering.

**unsupported assumption checked:** The attack requires a CEI violation in `PORTAL_TRADE_V2`. If that contract follows CEI (sets status before external call), the attack is blocked by the routing check alone. This is the key unverifiable assumption.

```text
    function _getSwapImplementation(address inputToken, address outputToken)
        internal
        view
        returns (address implementation)
    {
        // Determine if this is a buy or sell
        bool isBuy = _quoteTokenConfigurations[inputToken].enabled == 1;
        address baseToken = isBuy ? outputToken : inputToken;

        // Get token state to check status
        PackedTokenStateV2 memory state = _getTokenState(baseToken);

        if (state.status == TokenStatus.DEX) {
            // Token is listed on DEX, use PortalDexRouter
            return PORTAL_DEX_ROUTER;
        } else if (state.status == TokenStatus.Tradable) {
            // Token is still in bonding curve, use PortalTradeV2
            return PORTAL_TRADE_V2;
        } else {
            // Invalid or other status, will fail in implementation
            return PORTAL_TRADE_V2; // Let PortalTradeV2 handle the error
        }
    }
```

```text
    function _setTokenStatus(address token, IPortalTypes.TokenStatus status) internal {
        if (token != address(uint160(uint256(uint160(token))))) revert DirtyBits();
        uint256 slot;
        assembly ("memory-safe") {
            mstore(0x0, token)
            mstore(0x20, _packedTokenStates.slot)
            slot := keccak256(0x0, 0x40)
        }
        uint256 packed;
        assembly ("memory-safe") {
            packed := sload(slot)
        }
        uint8 header = uint8(packed);
        if (header != PACKED_TOKEN_STATE_HEADER) {
            packed = (packed & ~uint256(0xff)) | uint8(status);
            assembly ("memory-safe") {
                sstore(slot, packed)
            }
        } else {
            packed = (packed & ~(uint256(0xff) << 8)) | (uint256(uint8(status)) << 8);
            assembly ("memory-safe") {
                sstore(slot, packed)
            }
        }
    }
```
