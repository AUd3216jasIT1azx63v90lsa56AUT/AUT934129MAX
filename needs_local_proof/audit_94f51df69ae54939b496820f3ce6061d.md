## Attacker Path

**preconditions:**
- Portal is not halted (`bitFlags & 1 == 1`)
- `PORTAL_DEX_ROUTER` delegatecall target either (a) has no `DEX_ROUTER_MANAGER_ROLE` check, or (b) the check is bypassable
- A token with `TokenStatus.DEX` exists so that `swapExactInput` routes through `PORTAL_DEX_ROUTER`

**attacker-controlled inputs:**
- `token`: address of a legitimately launched DEX-listed token
- `poolInfo.pool`: attacker-controlled contract that mimics a DEX pool but redirects quote-token output to the attacker

**call sequence:**
1. Attacker calls `Portal.updateTokenPoolInfo(token, fakePoolInfo)` from an unprivileged EOA
2. If no role check exists in `PORTAL_DEX_ROUTER`, `_dexPools[token]` is overwritten with the fake pool
3. Any subsequent `swapExactInput` call on that token routes through `PORTAL_DEX_ROUTER`, which reads `_dexPools[token]` and calls the attacker's fake pool
4. The fake pool returns a manipulated output, draining quote tokens or BNB from Portal or from the swapper

## Why Existing Checks Fail

`onlyIfBitFlagsSet(CB_BIT_MASK_GLOBAL_SWITCH)` only checks that the global feature flag is on — it is not an `AccessControl` role check and does not restrict callers to `DEX_ROUTER_MANAGER_ROLE` holders. [6](#0-5) 

Whether a role check exists in the `PORTAL_DEX_ROUTER` delegatecall target **cannot be confirmed from the available source**, because that contract's source is not in the verified file set. The role constant `DEX_ROUTER_MANAGER_ROLE` appears only in the interface, not in any concrete implementation. If the delegatecall target omits the check, the entire access-control invariant for `_dexPools` is broken.

```text
    modifier onlyIfBitFlagsSet(uint256 mask) {
        if (!_checkBitFlags(mask)) revert FeatureDisabled();
        _;
    }

    /// @dev check bit flags
    /// If a bit is off, means the feature is disabled
    function _checkBitFlags(uint256 mask) internal view returns (bool) {
        // We only check the bits in the mask are set
        // And ignore the bits that are not in the mask
        //
        //  -  mask => with only checking bits set
        //  -  mask ^ bitFlags =>
        //          XOR: all checking bits should be unset, if
        //          they are set in the bitFlags.
        //  - _ & mask, ignore the bits that are not in the mask
        //
        return (mask ^ bitFlags) & mask == 0;
    }
```
