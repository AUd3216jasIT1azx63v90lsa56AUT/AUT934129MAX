## Why Existing Checks Fail

`Portal.sol`'s `swapExactInput` applies only `onlyIfBitFlagsSet(CB_BIT_MASK_GLOBAL_SWITCH)` before routing — no `SameToken` guard. [5](#0-4) 

`_getSwapImplementation` contains no `inputToken == outputToken` assertion. The `else` branch is explicitly designed to pass invalid states through to the facet with the comment "Let PortalTradeV2 handle the error." [6](#0-5) 

A grep across all indexed source files confirms `SameToken` appears **only** in `src/interfaces/IPortal.sol` — as an error declaration and one interface-level reference — and is absent from `Portal.sol` and `PortalBase.sol`. The check, if it exists at all, lives exclusively in the `PORTAL_TRADE_V2` facet, whose source is **not present in this repository**. [7](#0-6) 

`_getTokenState(address(0))` does not revert for `address(0)`: it reads `_packedTokenStates[address(0)] == 0`, interprets `header=0` as legacy format, and returns a zero-valued struct with `status = TokenStatus(0)`. No revert occurs in the dispatcher. [8](#0-7) 

## Rejection Checks
**expected behavior checked:** The NatSpec on `_getSwapImplementation` says "Invalid or other status, will fail in implementation" — this is the design intent, but it relies entirely on the facet to enforce `SameToken`. If the facet checks it before any value movement, the call reverts safely. If not, BNB movement is possible. [9](#0-8) 

**prior report checked:** No prior report found in indexed files.

**README/NatSpec checked:** No NatSpec on `swapExactInput` in `Portal.sol` documents a `SameToken` invariant at the dispatcher level.

**unsupported assumption checked:** The critical unknown is whether `PORTAL_TRADE_V2` enforces `SameToken` before any BNB transfer. The facet source is not in this repository. Without it, the impact (BNB drain vs. clean revert) cannot be determined from source alone.

```text
    function _getSwapImplementation(address inputToken, address outputToken)
        internal
        view
        returns (address implementation)
    {
        // Determine if this is a buy or sell
        bool isBuy = _quoteTokenConfigurations[inputToken].enabled == 1;
        address baseToken = isBuy ? outputToken : inputToken;

        // Get token state to check status
        PackedTokenStateV2 memory state = _getTokenState(baseToken);

        if (state.status == TokenStatus.DEX) {
            // Token is listed on DEX, use PortalDexRouter
            return PORTAL_DEX_ROUTER;
        } else if (state.status == TokenStatus.Tradable) {
            // Token is still in bonding curve, use PortalTradeV2
            return PORTAL_TRADE_V2;
        } else {
            // Invalid or other status, will fail in implementation
            return PORTAL_TRADE_V2; // Let PortalTradeV2 handle the error
        }
    }
```

```text
    function _getTokenState(address token) internal view returns (PackedTokenStateV2 memory state) {
        if (token != address(uint160(uint256(uint160(token))))) revert DirtyBits();
        uint256 packed;
        uint256 slot;
        assembly ("memory-safe") {
            mstore(0x0, token)
            mstore(0x20, _packedTokenStates.slot)
            slot := keccak256(0x0, 0x40)
            packed := sload(slot)
        }

        uint8 header = uint8(packed);
        if (header != PACKED_TOKEN_STATE_HEADER) {
            // Legacy: manually parse fields from packed uint256 (lower-order aligned)
            state.header = PACKED_TOKEN_STATE_HEADER;
            state.status = IPortalTypes.TokenStatus(uint8(packed));
            state.curveType = CurveType(uint8(packed >> 8));
            state.dexThreshType = DexThreshType(uint8(packed >> 16));
            state.id = uint32(packed >> 24);
            state.reserve = uint128(uint96(packed >> 56));
            state.circulatingSupply = uint128(uint96(packed >> 152));
            state.tokenVersion = IPortalTypes.TokenVersion(uint8(packed >> 248));
            state.quoteToken = IPortalTypes.QuoteTokenType.NATIVE_GAS_TOKEN;
            state.dexId = 0; // Default DEX ID for legacy tokens
            state.lpFeeProfile = IPortalTypes.V3LPFeeProfile.LP_FEE_PROFILE_LOW; // Default LP fee profile for legacy tokens
            state.feeProfile = FlapFeeProfile.FEE_GLOBAL_DEFAULT; // Default fee profile for legacy tokens
        } else {
            // parse slot0
            state.header = uint8(packed);
            state.status = IPortalTypes.TokenStatus(uint8(packed >> 8));
            state.curveType = CurveType(uint8(packed >> 16));
            state.dexThreshType = DexThreshType(uint8(packed >> 24));
            state.id = uint32(packed >> 32);
            state.tokenVersion = IPortalTypes.TokenVersion(uint8(packed >> 64));
            state.quoteToken = IPortalTypes.QuoteTokenType(uint8(packed >> 72));
            state.migratorType = IPortalTypes.MigratorType(uint8(packed >> 80));
            state.usingExtension = uint8(packed >> 88);
            state.dexId = uint8(packed >> 96);
            state.lpFeeProfile = IPortalTypes.V3LPFeeProfile(uint8(packed >> 104));
            state.feeProfile = FlapFeeProfile(uint8(packed >> 112));
            state.unused = uint136(packed >> 120);

            // read slot1
            uint256 slotValue;
            assembly ("memory-safe") {
                slotValue := sload(add(slot, 1))
            }
            state.reserve = uint128(slotValue);
            state.circulatingSupply = uint128(slotValue >> 128);

            // gas saving: only read slot2 if state.tokenVersion is not NATIVE_GAS_TOKEN
            if (state.quoteToken != IPortalTypes.QuoteTokenType.NATIVE_GAS_TOKEN) {
                // read slot2
                uint256 slot2Value;
                assembly ("memory-safe") {
                    slot2Value := sload(add(slot, 2))
                }
                state.quoteTokenAddress = address(uint160(slot2Value));
            }

            // read slot3 (extensionID) if token is using extension
            if (state.usingExtension != 0) {
                uint256 slot3Value;
                assembly ("memory-safe") {
                    slot3Value := sload(add(slot, 3))
                }
                state.extensionID = bytes32(slot3Value);
            }
        }
    }
```
