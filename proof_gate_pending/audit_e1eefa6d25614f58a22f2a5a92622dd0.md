## Attacker Path
**preconditions:**
- A token is in `TokenStatus.Tradable` with `circulatingSupply` just below `dexSupplyThresh`
- Attacker controls a receiver contract (e.g., a contract that receives BNB/quote-token refund or the minted token via `safeTransfer`)

**attacker-controlled inputs:**
- `params.inputToken = address(0)` (native BNB buy), `params.outputToken = <target token>`
- `params.inputAmount` sized to push `circulatingSupply >= dexSupplyThresh` and trigger migration

**call sequence:**
1. Attacker calls `swapExactInput` with a buy that crosses `dexSupplyThresh`
2. Inside `PORTAL_TRADE_V2` (delegatecall), the buy executes, `circulatingSupply` is updated, migration is triggered
3. **If** `luanchToDEX` (external call to migrator) is invoked **before** `_setTokenStatus(DEX)` is written, the token status is still `Tradable` during the external call
4. The migrator calls back into Portal (e.g., via a Uniswap V3 mint callback or a BNB `receive()` hook on an attacker-controlled address)
5. The reentrant `swapExactInput` call hits `_getSwapImplementation`: status is still `Tradable` → routes to `PORTAL_TRADE_V2`
6. Attacker buys additional tokens at bonding-curve price; DEX pool is not yet funded so the reserve/supply state is inconsistent

## Why Existing Checks Fail
**No reentrancy guard:** Grep across all `src/*.sol` files returns zero matches for `nonReentrant`, `ReentrancyGuard`, or any equivalent mutex. The only matches in the repo are in `lib/openzeppelin-contracts-upgradeable` and `lib/v3-core`, which are not applied to Portal's trade path.

**`_getSwapImplementation` status check is the only natural barrier**, but it only protects if `_setTokenStatus(DEX)` is written to storage **before** the external migrator call (CEI pattern). If the `PORTAL_TRADE_V2` implementation calls `luanchToDEX` first and sets status afterward, the status is still `Tradable` during the callback window, and `_getSwapImplementation` routes the reentrant call back to `PORTAL_TRADE_V2`.

**Critical gap:** The `PORTAL_TRADE_V2` implementation is a delegatecall target whose source is not in the indexed files (`src/Portal.sol`, `src/PortalBase.sol`, `src/PortalCommon.sol`). The exact ordering of `_setTokenStatus(DEX)` vs `luanchToDEX` cannot be confirmed from available source alone. [4](#0-3) 

## Rejection Checks
**expected behavior checked:** Buying at bonding-curve price after migration is triggered is not expected behavior; the invariant is that no bonding-curve trades occur after the DEX transition begins.

**prior report checked:** Not found in available source or NatSpec.

**README/NatSpec checked:** `IPortalMigrator` NatSpec states `luanchToDEX` "may be delegated called from a payable function" — confirming external call exposure. No reentrancy protection is documented. [5](#0-4) 

**unsupported assumption checked:** The attack requires either (a) the migrator making a callback to an attacker-controlled address (plausible for BNB-refund paths or V3 mint callbacks), or (b) the migrator itself being attacker-influenced. The V3 mint callback path (`uniswapV3MintCallback` / `pancakeV3MintCallback`) is a known re-entry surface in Portal-style launchers and is not an unsupported assumption.

```text
    function _getSwapImplementation(address inputToken, address outputToken)
        internal
        view
        returns (address implementation)
    {
        // Determine if this is a buy or sell
        bool isBuy = _quoteTokenConfigurations[inputToken].enabled == 1;
        address baseToken = isBuy ? outputToken : inputToken;

        // Get token state to check status
        PackedTokenStateV2 memory state = _getTokenState(baseToken);

        if (state.status == TokenStatus.DEX) {
            // Token is listed on DEX, use PortalDexRouter
            return PORTAL_DEX_ROUTER;
        } else if (state.status == TokenStatus.Tradable) {
            // Token is still in bonding curve, use PortalTradeV2
            return PORTAL_TRADE_V2;
        } else {
            // Invalid or other status, will fail in implementation
            return PORTAL_TRADE_V2; // Let PortalTradeV2 handle the error
        }
    }
```

```text
    /// @dev set the token status
    function _setTokenStatus(address token, IPortalTypes.TokenStatus status) internal {
        if (token != address(uint160(uint256(uint160(token))))) revert DirtyBits();
        uint256 slot;
        assembly ("memory-safe") {
            mstore(0x0, token)
            mstore(0x20, _packedTokenStates.slot)
            slot := keccak256(0x0, 0x40)
        }
        uint256 packed;
        assembly ("memory-safe") {
            packed := sload(slot)
        }
        uint8 header = uint8(packed);
        if (header != PACKED_TOKEN_STATE_HEADER) {
            packed = (packed & ~uint256(0xff)) | uint8(status);
            assembly ("memory-safe") {
                sstore(slot, packed)
            }
        } else {
            packed = (packed & ~(uint256(0xff) << 8)) | (uint256(uint8(status)) << 8);
            assembly ("memory-safe") {
                sstore(slot, packed)
            }
        }
    }
```
