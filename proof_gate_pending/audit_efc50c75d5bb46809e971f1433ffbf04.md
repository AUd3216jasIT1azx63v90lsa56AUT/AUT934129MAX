## Attacker Path
**preconditions:**
- A token is in `TokenStatus.Tradable` with `circulatingSupply` just below `dexSupplyThresh`.
- The migration path inside `PORTAL_TRADE_V2` makes an external call (to migrator/locker/router) **before** `_setTokenStatus(DEX)` is committed to storage.
- No reentrancy guard exists in the available source files (`nonReentrant` is absent from all `src/*.sol`). [4](#0-3) 

**attacker-controlled inputs:**
- A receiver contract that re-enters `swapExactInput` during the migrator callback.

**call sequence:**
1. Attacker calls `swapExactInput` with enough input to push `circulatingSupply` past `dexSupplyThresh`, triggering migration inside `PORTAL_TRADE_V2`.
2. `PORTAL_TRADE_V2` calls the external migrator/locker before writing `_setTokenStatus(DEX)`.
3. The migrator callback calls back into the attacker contract, which re-enters `swapExactInput`.
4. `_getSwapImplementation` re-reads `state.status` — still `Tradable` — and routes to `PORTAL_TRADE_V2` again.
5. Attacker buys tokens at bonding-curve price; `circulatingSupply` and `reserve` are updated a second time before the DEX pool is funded.

## Why Existing Checks Fail
The only guard visible in `Portal.sol` is the routing check in `_getSwapImplementation`, which reads `state.status` from storage. [5](#0-4) 

This check **would** block the re-entrant buy if `_setTokenStatus(DEX)` is written to storage **before** any external call in the migration path. However, if the ordering is reversed (external call first, status update second), the storage read still returns `Tradable` during re-entry and the gate passes. No `nonReentrant` modifier or equivalent lock is present in any of the in-scope source files to prevent this regardless of ordering. [6](#0-5) 

## Rejection Checks
**expected behavior checked:** Buying at bonding-curve price after migration is triggered but before the DEX pool is funded is not documented expected behavior; it violates the `Tradable→DEX` transition invariant.

**prior report checked:** No prior report found in the available repository files.

**README/NatSpec checked:** No NatSpec on `swapExactInput` or the migration path documents a reentrancy-safe ordering guarantee.

**unsupported assumption checked:** The critical assumption — that `_setTokenStatus(DEX)` is written **after** the external migrator call in `PORTAL_TRADE_V2` — cannot be confirmed or denied from the available source. `PORTAL_TRADE_V2` is not in `src/` and is not indexed. If the status is written first, the routing gate in `_getSwapImplementation` acts as a sufficient check and the attack fails. If the status is written after, the attack is live.

            revert FeatureDisabled();
        }

            let result := delegatecall(gas(), impl, 0, calldatasize(), 0, 0)
